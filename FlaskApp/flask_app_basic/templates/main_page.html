Map and Keep track of Chimpanzee Habitats!

chIMPRINT is an application that helps JGI to faster review, change, update and approve dynamic conservation maps as part of its collaborative Conservation Standards planning process with multiple stakeholders in different locations

Currently, the tool supports the following functionality:

  • Clone a Map
  • Draw/Edit shapes on the Map with notes/comments
  • Request the changes to be merged into another map
  • View/Approve requests for merging
